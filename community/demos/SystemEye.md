# System Design & Design Patterns Repository

Welcome to **Raj Gupta’s System Design & Design Pattern Notes** —  
a comprehensive collection of principles, UML diagrams, examples, and C++ implementations  
covering both **High-Level Design (HLD)** and **Low-Level Design (LLD)** concepts.

## Tools & Technologies

- **Language:** C++ (for implementation examples)  
- **Visualization:** Mermaid.js diagrams  
- **Docs Framework:** MkDocs + Material for MkDocs  
- **Icons:** Font Awesome / Material Icons
